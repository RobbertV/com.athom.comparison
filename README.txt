With these extra utilities you will enable more advanced features in your flow.
Your data is stored locally so you could use them in different flows or without the need of "Advanced Flows".

Always wanted to know how long your device is running or how much energy it used?
With this app you can compare these (or basically any kind of number) data in Homey.

Current features:
- Start/Stop duration (Calculate time in between)
- Start/Stop comparison with value - (Calculate difference of numbers)
- Convert number to currency
- Calculation with 2 numbers ([+] [-] [/] [*])
- Convert number to number with decimals
- Replace text in a sentence
- Create yes/no or number flow tags
- Trigger flow when duration and/or calculation is done
